## 1. 개요

V6.8의 핵심 목표는 **아티스트 연결고리(엣지)의 품질을 높이는 것**입니다.  
현재 2,633개 엣지 중 약 76%(2,019개)가 `INDIRECT`로, 이 중 상당수가 "동시대 음악적 파형 공유"라는 막연한 라벨로 연결되어 있어 실질적인 관계 정보로서 가치가 떨어집니다.

## 2. Task A: 막연한 GENRE_OVERLAP 엣지 가중치 하향 조정

### 현재 문제점

| 라벨 | 개수 | 현재 가중치 | 문제 |
|------|------|-----------|------|
| 동시대 음악적 파형 공유 | 858개 | 0.40 | 장르+시대 겹침만으로 자동 생성. 실질적 교류 근거 없음 |
| 시대를 관통하는 K-Pop 무드 | 602개 | 0.40 | K-Pop 풀에서 랜덤 연결. 근거가 더욱 희박 |

### 개선 작업

1. `v6.5-socialize-lonely-stars.ts` 스크립트에서 생성하는 GENRE_OVERLAP 엣지의 기본 weight를 **0.4 → 0.15**로 하향
2. 빌드 스크립트(`v5.4-build-universe.ts`)에서 GENRE_OVERLAP → INDIRECT 변환 시 weight를 **0.1~0.2** 범위로 클램프
3. `GraphCosmos.tsx`의 엣지 렌더링에서 weight < 0.2인 엣지는 **opacity 10% 이하**로 거의 보이지 않게 처리
4. 라벨을 "동시대 음악적 파형 공유" → "장르 유사성" 등 더 솔직한 표현으로 교체

### 기대 효과

- 실질적인 관계(협업, 프로듀싱, 작곡, 그룹 등)의 연결선이 시각적으로 더 도드라짐
- 막연한 연결선이 배경 소음(noise)처럼 거의 보이지 않게 됨
- 유저가 "이 두 아티스트가 왜 연결되어 있지?"라는 의문을 덜 갖게 됨

## 3. Task B: 컴필레이션 앨범 기반 크롤링 및 엣지 생성

### 개념

같은 컴필레이션 앨범(예: "No Alternative", "사랑도 통역이 되나요 OST", "EBS 스페이스 공감" 등)에 참여한 아티스트들은 **동시대에 실제로 같은 프로젝트에 소속되었던** 명확한 관계가 있으므로, 이를 `COMPILATION` 엣지로 연결합니다.

### 데이터 소스

**MusicBrainz API** — 아티스트가 참여한 앨범(release) 조회:
```
GET /ws/2/release?artist={mbid}&type=compilation&fmt=json&limit=100
```

각 release의 `release-group.id`가 같으면 → 같은 컴필레이션에 참여한 것

### 크롤링 파이프라인

```
[Step 1] 우주에 있는 989명의 아티스트 MBID 목록 추출
    ↓
[Step 2] 각 아티스트에 대해 MusicBrainz API로 compilation release 조회
    ↓ (Rate Limit: 1 req/sec)
[Step 3] release_group_id 별로 참여 아티스트 그룹핑
    ↓
[Step 4] 같은 release_group에 2명 이상 참여한 경우 → COMPILATION 엣지 생성
    ↓
[Step 5] organic-graph.json에 엣지 주입
    ↓
[Step 6] v5.4-build-universe.ts 재빌드
```

### 엣지 생성 규칙

| 항목 | 값 |
|------|---|
| relation | `COMPILATION` |
| weight | **0.5** (실제 같은 앨범 참여이므로 GENRE_OVERLAP보다 높음) |
| label | 컴필레이션 앨범명 (예: "No Alternative 참여", "사랑도 통역이 되나요 OST") |

### 예상 규모

- 989명 × 1 req/sec = 최대 **~17분** 크롤링 소요
- 한국 음악 씬의 컴필레이션 참여는 빈번하므로 **수백 개**의 새 엣지가 생성될 것으로 예상
- 기존의 막연한 GENRE_OVERLAP을 대체할 수 있는 **높은 품질의 관계 데이터**

### UI 표현

| 엣지 유형 | 색상 | 스타일 |
|----------|------|--------|
| `COMPILATION` | `#f472b6` (핑크) | 실선, opacity 0.6 |
