# 🌟 K-Culture Universe V7.5 Idea Sketch: 다차원 관계망 확장 (Multi-dimensional Relations)

> **작성일**: 2026-03-22
> **상태**: 💡 아이디어 구상 단계 (Idea Sketch)
> **배경**: V7.0.x 시리즈에서 해결한 '아티스트 데이터 정합성'을 바탕으로, 아티스트 간의 연결 고리(엣지)를 더 풍부하고 현실세계와 가깝게 확장하기 위한 미래 계획입니다.

---

## 1. 개요 (Overview)

현재 우주는 음악적 협업(`FEATURED`)과 장르/플레이리스트 유사성(`GENRE_OVERLAP`)을 중심으로 연결되어 있습니다.
V7.5에서는 **가족(혈연), 학연, 지연, 소속사, 오프라인 이벤트 동반 참석** 등 현실 세계의 다양한 인연을 우주에 반영하여, 더 흥미롭고 예측 불가능한 발견을 제공합니다.

---

## 2. 신규 관계(Edge) 타입 및 가중치 (Weight) 설계

당기는 힘(Weight, 0.0 ~ 1.0)을 차등 적용하여, 정말 가까운 사이는 우주에서도 찰떡같이 붙어있게 만듭니다.

| 관계 타입 (Relation) | 설명 및 예시 | 제안 가중치 |
|----------------------|--------------|-------------|
| **`FAMILY`** | **가족 / 혈연 / 혼인**<br>ex) 이상순 - 이효리 (부부)<br>ex) 신중현 - 신대철 (부자)<br>ex) 이찬혁 - 이수현 (남매) | **1.0** (완전 결속) |
| **`FEATURED`** | **정식 음원 협업 (기존)**<br>ex) 아이유 - 슈가 (에잇) | **0.7** (강한 결속) |
| **`ALUMNI_INTIMATE`** | **동기 동창 / 친한 동문**<br>ex) 고등학교 같은 반, 같은 기수 연습생 | **0.7** |
| **`AGENCY_MATE`** | **같은 기획사 / 레이블 소속**<br>ex) 뉴진스 - 르세라핌 (HYBE)<br>ex) 안테나 뮤직 식구들 | **0.6** |
| **`ALUMNI`** | **일반 동문 / 학연**<br>ex) 같은 학교 선후배 (서공예, 한림예고 등) | **0.3 ~ 0.4** |
| **`NEIGHBOR`** | **같은 동네 / 지연**<br>ex) 같은 지역 출신으로 친목이 확인된 경우 | **0.3** |
| **`EVENT_CO`** | **같은 행사 / 페스티벌 동반 참석**<br>ex) 펜타포트 록 페스티벌 2023 헤드라이너 | **0.3** |
| **`NEWS_MENTION`** | **기사 동시 언급**<br>ex) 같은 음악 트렌드 기사에 긍정적으로 묶여서 언급됨 | **0.15** |

## 3. 데이터 검증 및 수집 시스템 (User-Driven)

음악 콜라보와 달리, 가족이나 학연, 행사 참석 여부는 자동 크롤링이나 LLM에 전적으로 의존하기 위험합니다 (환각/가짜 스캔들 우려). 
따라서 **참여형 검증 시스템**을 도입합니다.

### 3.1 유저 제안 시스템 (Edge Proposal)
사용자가 우주를 탐험하다가 "이 두 아티스트는 부부인데 연결이 안 되어 있네?" 라고 생각할 때:
1. 두 아티스트를 선택하고 **"관계 제안하기"** 버튼 클릭
2. 관계 유형 선택 (Dropdown: 가족, 같은 기획사, 학연 등)
3. **증빙 링크 제출 (필수)**: 네이버 뉴스, 나무위키, 공식 기사 URL 입력 (거짓 데이터 방지용)

### 3.2 관리자 승인 시스템 (Admin Verification)
1. `/admin/relations` 대시보드에서 유저 제안 목록 확인
2. 관리자가 첨부된 기사 링크(`reference_url`)를 클릭하여 사실 확인
3. `[승인]` 클릭 시, 해당 엣지가 우주의 `layout.json`에 즉시 가중치와 함께 반영됨

## 4. UI/UX 기대 효과

* **관계선(Edge) 시각화 다양화**: 엣지 타입에 따라 선의 색상이나 애니메이션을 다르게 표현. (예: `FAMILY`는 빛나는 붉은색 선, `AGENCY_MATE`는 소속사 고유 색상의 선)
* **스토리텔링 툴팁**: 엣지에 마우스를 올렸을 때 "부부 관계", "2018년 지산 록 페스티벌 동반 출연" 등의 텍스트가 표시되어 탐험의 재미 극대화.
* **우주 지형의 진화**: 음악적 유사성이 전혀 없는 두 아티스트라도, 같은 레이블이거나 가족이면 우주 한가운데 웜홀처럼 서로를 강하게 끌어당기는 기이하고 재미있는 우주 지형도 형성.

> **Next Steps (미래 도입 시)**: 
> 1. `V5Edge` 타입에 위 열거형(Enum) 관계들 추가 (`src/lib/types.ts`)
> 2. Supabase에 `edge_proposals` (제안, 링크, 상태 보유) 테이블 신설

## 5. 💜 하트 투표 시스템 (Heart Voting)

> **한 줄 요약**: 정회원이 하루 100하트를 사용하여 아티스트 간 관계의 거리를 직접 조정할 수 있는 팬덤 참여 시스템

### 컨셉
- **정회원(Full Member)** 에게 하루 100하트 제공 (자정 KST 리셋)
- 두 아티스트를 선택한 후 "더 가깝게 💜" 또는 "더 멀게 💔" 버튼으로 하트를 소비
- 하트 투표는 해당 엣지의 가중치(weight)에 미세한 가산/감산 (ex: ±0.001/하트)을 적용
- 일정 투표수 이상 누적 시 실제 우주 레이아웃에 반영 → 팬덤의 힘으로 우주 지형이 변화

### 기대 효과
- **팬덤 유입·리텐션**: "내 아티스트를 우주 중심으로!" 같은 커뮤니티 이벤트 가능
- **자연스러운 데이터 보강**: 팬들이 실제로 아는 관계를 투표로 반영

### 악용 방지
- 하루 100하트 제한 + 같은 쌍에 하루 최대 10하트
- 급격한 가중치 변화 방지: 1일 최대 ±0.05까지만 반영
- 관리자 대시보드에서 이상 패턴(봇 의심) 모니터링

## 6. 🚀 "~~~로부터 ~~~에게" 여정 공유 링크

> **한 줄 요약**: 두 아티스트 사이를 우주를 유영하며 미리듣기로 이동하는 여정을 링크로 공유

### 컨셉
- 사용자가 출발 아티스트(A)와 도착 아티스트(B)를 지정
- 시스템이 A→B 최단 경로(또는 추천 경로)를 계산
- 경로 상의 각 아티스트 노드를 차례로 방문하며 미리듣기(previewUrl) 자동 재생
- 이 여정이 고유 URL로 생성됨: `kcultureverse.com/from/[A_id]?to=[B_id]`
- 링크 수신자에게 "**아이유로부터 뉴진스에게**" 같은 감성적인 타이틀과 함께 OG 카드 표시

### 기대 효과
- **바이럴 공유**: SNS에 "나만의 우주 여정" 링크 공유 → 자연스러운 유입
- **음악 발견**: 중간 경유 아티스트의 미리듣기를 통해 몰랐던 음악을 발견
- **감성적 UX**: "내가 좋아하는 아티스트에서 네가 좋아하는 아티스트까지" 선물 느낌

## 7. 🎨 아티스트별 개인화 우주 렌더링 (Personalized Universe View)

> **한 줄 요약**: 아티스트에 따라 우주의 색상, 형태, 분위기가 달라지는 개인화 뷰

### 컨셉
- 특정 아티스트를 중심으로 바라볼 때, 그 아티스트의 음악 세계를 반영한 색상/형태로 우주가 렌더링됨
- **알고리즘 후보**:
  - 앨범 아트 대표 색상 추출 (color quantization) → 우주 배경 그라디언트
  - 장르/무드 키워드 → 형태 매핑 (발라드=부드러운 곡선, 록=날카로운 파티클)
  - 팬덤 컬러(공식 응원봉 색상) → accent 오버라이드
- "이 아티스트의 팬덤이 바라본 음악 세상은 이런 모습이다" 라는 감성적 체험 제공

### 기대 효과
- 같은 우주를 보더라도 **누구를 중심으로 보느냐에 따라 전혀 다른 경험** 제공
- 팬덤별 스크린샷 공유 → 자연스러운 바이럴

## 8. 🎤 아티스트 공인 인증 시스템 (Verified Artist)

> **한 줄 요약**: 아티스트 공식 계정이 DM으로 요청하면 인증 배지 부여 + 커스터마이즈 허용

### 절차
1. 아티스트 본인 또는 소속사가 공식 SNS 계정 (@sloe_pangyo 등)으로 DM 발송
2. 운영자가 공식 계정 여부 확인 후 `verified: true` 플래그 부여
3. 우주에서 해당 아티스트 노드에 ✅ 인증 배지 표시

### 아티스트 혜택
- **커스텀 링크 추가**: 신보 발매 링크, 콘서트 예매 링크 등을 노드에 직접 첨부 가능
- **프로필 수정 요청**: 이름 표기, 이미지, 장르 등의 수정 요청 우선 처리
- **홍보 채널 활용**: "K-Culture Universe에서 나를 찾아보세요" 등의 마케팅에 활용 유도

### 기대 효과
- 아티스트 본인이 참여하는 플랫폼 → 신뢰도 및 공신력 대폭 상승
- 아티스트의 자발적 홍보 → 팬덤 유입 극대화
