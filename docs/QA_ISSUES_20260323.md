## 🔴 미해결 이슈 목록

### #1 우주맵 아티스트 사진 안 나옴
- **현상**: MID 줌 레벨에서 모든 노드가 이니셜 원으로만 표시됨
- **실제 원인 (확인 완료)**:
  1. MID LOD에서 이미지 렌더 코드 자체가 없었음
  2. CLOSE LOD에서 `isFocused || isHop1 || isMajor` 조건이 과도하게 엄격
  3. **핵심**: `getCachedImage()`가 `new Image()` 로드 후 `onload` 콜백이 없어서 이미지가 로드돼도 Canvas가 재렌더되지 않음
  4. `IMG_CACHE_MAX = 50` 으로 756개 이미지 중 50개만 캐시 (나머지 매번 재로드)
- **코드 수정 완료** (커밋 `2015a5b`) — Vercel 배포 후 확인 필요
  - MID LOD에 이미지 렌더 추가
  - CLOSE LOD 조건 제거
  - `onload → fg.refresh()` 추가
  - `IMG_CACHE_MAX` 50 → 300

### #2 카드덱(바텀시트) 하단 빈 공간

- **원인**: `height: "auto"` + framer-motion spring 조합이 실제로 shrink-wrap 안 됨
- **코드 수정 완료** (커밋 `2015a5b`) — Vercel 배포 후 확인 필요

### #4 엣지(관계선) 클릭 → 설명 팝업 안 나옴

- **원인**: `onLinkClick` 내 `if (!focusedId) return` 조건으로 포커스 아티스트 없으면 완전 차단됨
- **코드 수정 완료** (커밋 `2339354`) — Vercel 배포 후 확인 필요
  - `focusedId` 없어도 팝업 허용, 있으면 1촌 엣지만 허용
- **추가 의심 원인**: UI 레이어(`pointerEvents`) 문제일 수도 있음 → 배포 후 미해결이면 추가 조사
