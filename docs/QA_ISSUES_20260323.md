## 🔴 다음 세션 이관 이슈 목록 (Pending Issues)

### #1 바텀시트(카드덱) UI 정렬 및 여백 이슈
- **현재 상태**: 높이 계산 방식과 쓸데없는 여백(padding/margin)을 1차적으로 줄이는 코드가 배포되었음.
- **문제점**: 화면에서 여전히 카드가 너무 작게 보이거나, 버튼 등이 어긋나 보이는 등 UI 레이아웃이 여전히 "엉망"으로 느껴지는 문제가 잔존.
- **다음 작업**: `BottomSheet.tsx`, `MiniPlayer.tsx`, `warp-list` CSS 등을 브라우저 개발자 도구로 정밀 수치 조절 및 반응형(vw, vh) 테스트 필요.

### #2 수동 미리듣기 이어재생 (자율주행 유사 기능)
- **현재 상태**: 기능 미구현 상태. 현재 클릭 시 30초만 재생 후 정지됨. (원래 자율주행 `isWarping` 전용 기능이었음)
- **문제점**: 사용자는 노래가 끝나면 다음 곡으로 넘어가는(또는 같은 아티스트의 다음 곡으로) 동작을 기대함.
- **다음 작업**: `MiniPlayer`와 오디오 재생 컨텍스트(`useAudio` 또는 백그라운드 관리 훅)를 연동하여 트랙 종료 시 다음 큐로 넘기는 로직 신규 구현.

### #3 엣지(관계선) 클릭 팝업 이슈 지속 확인
- **현재 상태**: 포커스 유무와 상관없이 클릭 가능하게 코드는 수정 및 배포됨.
- **문제점**: Vercel 배포 후에도 여전히 클릭 시 설명 팝업이 안 뜬다면, 이벤트 감지(pointerEvents)나 z-index 문제일 가능성 높음.
- **다음 작업**: `GraphCosmos.tsx`의 엣지 렌더 영역 및 `onLinkClick` 콜백 작동 여부를 콘솔 로그로 추적.
