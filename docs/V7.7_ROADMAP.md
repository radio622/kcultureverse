# 🌟 K-Culture Universe V7.7 개발 로드맵 (Roadmap)

> **작성일**: 2026-03-23
> **목적**: V7.4/V7.5 세션의 잔여 QA 이슈(버그)를 최우선으로 해결하고, V7.5 아이디어 스케치에서 착수하지 못했던 혁신적인 UI/UX 기능들을 구체적인 순서로 현실화하기 위한 V7.7 종합 릴리즈 로드맵입니다.

---

## 📅 Phase 1: 치명적 시각 및 데이터 버그 조기 진압 (Blockers & Hotfixes)
가장 시급하게 해결해야 할 우주의 렌더링 결함과 데이터 인프라 봇의 중단 사태를 먼저 해결합니다.

### 1-1. 🌌 맵 상 엣지(관계선) 실종 버그 복구 (QA 해결)
- **상태**: 현재 아티스트 노드만 보이고 서로를 잇는 엣지 실선이 투명하거나 렌더링 누락된 상태. (최고 심각도)
- **액션**: `GraphCosmos.tsx`의 `linkColor`, Canvas Line 렌더링 로직, LOD 조건 점검 및 투명도 롤백. 선이 시각적으로 선명하게 연결되도록 복원.

### 1-2. 🤖 듀얼 하베스터 봇 (Spotify 401 에러) 패치 및 재구동 (QA 해결)
- **상태**: 스크립트 진행 중 토큰 만료 갱신 부재로 44번째 아티스트에서 중단(kill)됨.
- **액션**: 
  - `v7.5-dual-harvester-draft.py`에 Spotify Client Credentials '토큰 리프레시' 흐름 추가.
  - `scripts/progress.json`의 체크포인트를 읽고 45번째 아티스트(A.TRAIN 등)부터 이어하기(Resume) 구현.
  - 무중단 24시간 백그라운드 크롤링 안전성 확보.

### 1-3. 👆 엣지(관계선) 클릭 팝업 히트박스 교정 (QA 해결)
- **상태**: 엣지 클릭이 뒤쪽 노드로 뚫고 들어가 잘못 잡히는 이슈.
- **액션**: `GraphCosmos.tsx`의 엣지 Hitbox 두께 값 전역 조정 및 Raycaster 이벤트 우선순위 변경하여 의도한 선이 바로 클릭되도록 조치.

## 📅 Phase 2: 청취 몰입감과 UI/UX 폴리싱 (Polishing)
앱의 '사용성'을 최상단으로 끌어올리는 잔여 프론트엔드 최적화 단계입니다.

### 2-1. 📻 수동 미리듣기 '라디오식 연속 재생' 큐 시스템 구축 (QA 요구사항)
- **상태**: 30초 재생 후 멈춤. 유저가 바텀시트를 누를 때 맥이 끊김.
- **액션**: 바텀시트에서 카드 수동 클릭 시에도, 30초가 끝나면 다음 추천 곡 혹은 연관 아티스트 카드로 자동 워프(자율주행 유사)되도록 `MiniPlayer.tsx`와 `useAutoWarp` 오디오 로직 융합.

### 2-2. 📱 바텀시트 하단 UI 파편화 리팩토링 (QA 해결)
- **상태**: 고정 px 여백 제거에도 불구하고 기기 해상도에 따라 카드 덱 안의 버튼, 제안 버튼 정렬이 깨지는 현상.
- **액션**: `BottomSheet.tsx`의 절대치 px를 걷어내고, 반응형 상대 치수(vh, vw, %) 및 CSS Grid/Flexbox 기반의 견고한 컴포넌트 틀로 대공사 재설계.

## 📅 Phase 3: V7.7의 혁신, 신규 바이럴 및 지형지물 확장 (Feature Core)
여기서부터 새롭게 추가되는 V7.5 아이디어의 핵심 킬러 기능들을 개발합니다. 이 중 바이럴 파급력이 가장 쎈 '여정 링크'를 최우선 개발합니다.

### 3-1. 🚀 "A로부터 B에게" 우주 여정 자동 재생 및 OG 공유
- **개념**: "아이유로부터 뉴진스에게" 등, 출발/도착 아티스트 간 최단 경로 웜홀 플레이리스트를 유영하며 연속 재생하는 기능.
- **액션**:
  - `from`, `to` 쿼리 파라미터 라우팅 화면 개발 (`/from/[A]?to=[B]`).
  - 예쁜 시각적 OG 썸네일 자동 생성 및 SNS 공유(바이럴) 클립보드 UI 구축.

### 3-2. 🌐 다차원 관계망 (가족, 학연, 소속사) 엣지 확장 및 유저 검증
- **개념**: 단순 음악 콜라보(`FEATURED`)를 넘어 가족(`FAMILY`, 1.0), 소속사(`AGENCY`, 0.6) 등의 현실 엣지(관계)를 표현.
- **액션**:
  - 유저가 "관계 제안하기" 폼을 통해 증거(기사 URL 등)와 함께 제안하는 시스템 개발.
  - 관리자(`/admin`) 대시보드 검토/승인 및 V5Edge 타입 확장 업데이트.

### 3-3. 🎨 아티스트별 앨범/테마 연동형 우주 개인화 렌더링
- **개념**: 특정 아티스트 포커스 시, 우주의 바탕색과 렌더링 입자들이 해당 앨범/가수의 테마색과 분위기에 맞춰 변화.
- **액션**:
  - 이미지 퀀타이제이션 알고리즘으로 앨범 커버의 Main Color 추출.
  - 추출된 색상을 우주 Canvas 글로벌 배경색(그라데이션 및 파티클 색상) 객체에 동적 바인딩.

### 3-4. 💜 정회원 엣지 '하트 투표' 물리적 지형 변화
- **개념**: 정회원이 하루 100개씩 좋아하는 두 노드 사이의 거리에 투표("가깝게!" / "멀게!")하여 실제 엣지 가중치(`weight`)가 수정되는 팬덤의 장.
- **액션**:
  - Supabase `heart_votes` 테이블 설계 및 투표 API 누적 로직 구현.
  - 매일 자정 또는 누적 티핑포인트 도달 시 `layout.json` 상의 거리를 변화시키는 CRON 배치 구축.

### 3-5. 🎤 아티스트 공인 인증(✅) 배지 및 커스텀 마케팅 링크
- **개념**: 가수 본인 요청 시 오피셜 배지 획득 및 노드 내 콘서트 단독 예매 링크 삽입 기능.
- **액션**: 노드 뱃지 렌더러 추가 및 JSON 페이로드에서 `verified_links` 배열 확장. (기획 및 운영 정책 단 우선 수립)
